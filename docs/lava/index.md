Geyser Lava是基于Geyser的任务库，通过Geyser Lava能够提高开发效率，提供基础的辅助任务，并包括对PyTorch等简单支持。

## 支持

- 运行时基础操作

- PyTorch

对其他领域任务的低代码实现，欢迎开发者对能够复用的任务进行补充！
